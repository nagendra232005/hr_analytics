# hr_analytics
# HR Analytics Dashboard using Power BI

## 📌 Project Overview
This project presents an interactive HR Analytics Dashboard developed in Power BI to help HR teams monitor employee performance, attendance, promotions, awards, projects, salaries, and attrition. The dashboard provides actionable insights that support workforce planning and decision-making.

## 🎯 Objectives
- Analyze employee performance and productivity.
- Monitor attendance and overtime.
- Track promotions and employee awards.
- Analyze project allocation across departments.
- Identify employee attrition trends.
- Provide HR with data-driven insights through interactive dashboards.

## 📊 Dashboard Features
- Employee Overview
- Department-wise Employee Distribution
- Attendance Analysis
- Performance Analysis
- Salary Analysis
- Promotion Analysis
- Awards Analysis
- Project Analysis
- Attrition Analysis
- Interactive Filters and Slicers

## 📈 Key Insights
- Highest Attrition Department: Sales
- Best Attendance Month: January
- Lowest Performance Category: Need Improvement
- Top Salary: ₹70,700
- Average Overtime: 6 Hours
- Highest Promoted Department: IT
- Most Awarded Department: IT

## 🛠 Tools & Technologies
- Power BI Desktop
- Microsoft Excel
- Power Query
- DAX
- Data Modeling

## 📂 Dataset
The dataset includes:
- Employees
- Departments
- Attendance & Leave
- Performance
- Payroll
- Promotions
- Projects
- Awards
- Engagement
- Health & Wellness
- Attrition

## 🔗 Data Model
The dashboard uses a Star Schema with the **Employees** table as the central table connected to related HR tables using one-to-many relationships.
